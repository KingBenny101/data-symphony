# This is the Backend
The backend was made using Django REST Framework

## About the API
> Due to some errors I wasn't able to run the app with ```DEBUG = False```
### Overview
- http://127.0.0.1:8000/main/
- Shows the overview of the API
### List
- http://127.0.0.1:8000/main/list/
- Lists all the entries
### Create
- http://127.0.0.1:8000/main/create/
- Creates a new entry
### Delete
- http://127.0.0.1:8000/main/delete/<id>
- Deletes the specified entry
### Update 
- http://127.0.0.1:8000/main/update/<id>
- Updates the specified entry
### View
- http://127.0.0.1:8000/main/view/<id>
- Returns JSON data of the entry
### Report 
- http://127.0.0.1:8000/main/report/<id>
- Generates report for the specified entry
